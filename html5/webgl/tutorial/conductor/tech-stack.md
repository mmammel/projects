# Tech Stack

## Frontend
- **HTML5 & CSS3:** For the application's structure and responsive styling.
- **JavaScript:** Core programming language for application logic and user interaction.
- **WebGL:** High-performance 3D graphics API used for hardware-accelerated rendering of geometric models.

## Libraries & Frameworks
- **jQuery (2.2.0):** Facilitates DOM manipulation, event handling, and asynchronous API communication.
- **Sylvester.js:** Provides robust vector and matrix math operations necessary for 3D coordinate transformations.

## Tools & Utilities
- **Java:** Used for specialized offline data processing tasks, such as vertex mapping and triangle generation.
- **Shell Scripting (Bash):** Employed to automate data generation and transformation workflows.
