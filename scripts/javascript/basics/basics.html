All programming languages are different, but also very similar. The basic bulding blocks of all laguages can be seen and practiced with javascript, which is the easiest language to play around with because all you need is a text editor and browser. To run some javascript in your browser that you wrote, start with a file like this: 

        <html>
          <head>
            <script type="text/javascript">
              // this is where you write your functions, and call them.
            </script>
          </head>
          <body>
          </body>
        </html>

In between those script tags, you can write functions, and then call them. For example: 

        <html>
          <head>
            <script type="text/javascript">
              // define a function
              function doAnAlert() {
                alert( "I made an alert");
              }
           
              doAnAlert(); // now call it
            </script>
          </head>
          <body>
          </body>
        </html>

Put this text in a file, name it something like "myjavascript.html", then in a browser go to:

(On a Mac): "file:///Users/username/myjavascript.html"
(On windows): "file://C:/Users/username/myjavascript.html"

And it will run in the browser. That's it, pretty easy. w3schools is a great resource for javascript (and many other languages), it is run by the w3 council (or something like that) - it's the standards body that agrees on what everyone should agree a certain language should support, because if everyone agrees we can all write cool stuff with it.
